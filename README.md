# PlayNextTab-Privacy-Policy
PlayNextTab Privacy Policy (Chrome Extension)


This extension does not collect sensitive user data. It temporarily accesses open YouTube video tabs to enable playlist functionality and stores only your preferences (auto-switch/close settings) locally. No personal information is ever shared or transmitted.

All permissions are strictly required for the extension’s single purpose: automatically managing YouTube video tabs as a playlist.

No broad host permissions (e.g., *://*/*) or invasive APIs (e.g., bookmarks, history) are used.

This configuration adheres to Chrome Web Store’s Permission requirements. 

Privacy Notes:

- No external servers: All data stays on the user’s device.

- No tracking: No cookies, analytics, or hidden data collection.

- YouTube thumbnails: Loaded directly from YouTube’s servers (not stored).


WHAT THE EXTENSION DOES USE:

YouTube Video Metadata -Titles/URLs of open YouTube tabs (to generate thumbnails and "Next Up" info). Only temporarily, cleared when tabs close.
